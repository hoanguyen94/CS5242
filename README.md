# Project Workflow (Local Editing + Colab GPU)

This project is designed for an efficient workflow where:

- Code is **edited locally** using an IDE (e.g. VSCode)
- Training runs on **Google Colab GPU**
- Stable code is **committed to GitHub**

---

# How to run notebooks using Google Colab runtime in your VScode

- Install Colab Extension
- Connect to Colab runtime in your VSCode
(For more details, follow this instruction: https://developers.googleblog.com/google-colab-is-coming-to-vs-code/)
